cyclotomic polynomials

Cyclotomic polynomial theory is supposedly “advanced” in the realm of number theory. Let’s see.

Cyclotomic polynomials have multiple definitions, and the easiest one to remember is that

∏d∣nΦd(x)=xn–1∏d∣nΦd(x)=xn–1
which is a bit yeah. Hmm. If that definition didn’t make much sense then let’s try a different one: the nnth cyclotomic polynomial is the unique irreducible integer polynomial factor of xn−1xn−1 which is coprime with all xk−1xk−1 for k<nk<n.

An example is Φ6(x)=x2−x+1Φ6(x)=x2−x+1, as x2+x+1x2+x+1, x−1x−1, x+1x+1 are not coprime with smaller xk−1xk−1 things.

If that didn’t make sense then I’d suggest the wikipedia page for cyclotomics. If that still doesn’t make sense, then in all honesty try something else for now.

Also, before we move on, make sure you’re acquainted with mods.

Anyways, the cool thing about cyclotomics is that if p|Φk(n)p|Φk(n) then either p∣kp∣k or p≡1modkp≡1modk. Pretty nifty, hey? Try proving this later on; it’s definitely worth doing.

Let’s give it a shot:

Prove that x2+9=y3x2+9=y3 has no positive integer solutions.

First look at the equation mod 4. Note that this means that xx has to be even, and that yy has to be 11 mod 4.

Rearrange the equation to get x2+1=(y−2)(y2−2y+4)x2+1=(y−2)(y2−2y+4). Notice that since the LHS is just Φ4(x)Φ4(x), its prime factors are either 22 or 11 mod 44. But yy is 11 mod 4 which instantly dies.
