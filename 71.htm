“Divide and conquer” – not me

When aa is a factor of bb, we write a∣ba∣b. Formally, this means that there is some integer kk such that b=akb=ak.

(if you’re wondering where the strikethrough guy and the italics guy are, I disposed of them yesterday)

For instance, 3|20193|2019 because 3×673=20193×673=2019 but 4∤20194∤2019 because you can’t find an integer kk such that 4×k=20194×k=2019. One last note — in general, you don’t actually need to find such a kk. Just saying that kk exists is enough.

Some basic things about divisibilities, which you should prove (they’re not that hard)

  1. If a∣ba∣b and b∣cb∣c then a∣ca∣c
  2. If a∣ba∣b then a∣b+kaa∣b+ka for any integer kk
  3. If a∣ba∣b and c∣dc∣d then ac∣bdac∣bd
  4. If a∣ba∣b then |a|≤|b||a|≤|b|

Now that we have that sorted, it’s time for some easy exercises.

Note that from 2, we can subtract or add any multiple of n2+1n2+1 from the RHS. So we subtract n(n2+1)n(n2+1), and we get n2+1∣2n2+1∣2, which means that n2+1n2+1 is just 11 or 22.

This one looks hard, but all you have to note is that m∣mn3–m2nm∣mn3–m2n, so we have m∣m4n–m3+m2n9+2019m∣m4n–m3+m2n9+2019, and everything except 20192019 is a multiple of mm, so m∣2019m∣2019. It’s easy to finish now (if you know how to factorise 2019, of course.)

Removing a single factor from the left hand side is often really useful for that type of problem.

A well known theorem is as follows:

Fundamental Theorem of Arithmetic

Every positive integer has a unique (up to the order of factors) prime factorisation.

For instance, 120 = 2 x 2 x 2 x 3 x 5. Simple.

You can use the prime factorisation to work out cool things, like the number of factors a number has, or the Euler totient function. For factors, write the number in using exponents (in the form ∏paii∏piai), take all of the exponents (and ignore the primes), add one to each, and add together.

One more technique – it’s often useful to consider how many times a prime divides something. We call this νpνp, or “p-adic valuation”. Basically, νp(x)νp(x) is the exponent of pp in the prime factorisation of xx, which sounds really complicated, but if you look at some examples it’ll make sense:

ν2(4)=2,ν3(10)=0,ν5(1000)=3,ν2(4)=2,ν3(10)=0,ν5(1000)=3, etc.

Find all positive integers nn such that 22n∣nn.22n∣nn.

Note that ν2(22n)=2nν2(22n)=2n, and ν2(nn)=nν2(n)ν2(nn)=nν2(n). So if ν2(n)≥2ν2(n)≥2, nn works, i.e. nn is any multiple of 44.
