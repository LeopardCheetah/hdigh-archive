combinatorial induction

Let’s do induction, except let’s do it properly. What is this “induction” thing?

Suppose you want to prove that a statement is true for every positive integer nn. Proving it one at a time, starting at n=1n=1, then proving it true for n=2n=2, and so on, would take a long time. Forever, in fact. It’s just not physically possible to do it.

A better idea is, instead, to prove it true for n=1n=1. Then you prove that if it’s true for n=kn=k, it’s true for n=k+1n=k+1. This is the hard part about induction, and the step that takes the most effort to understand. Let’s unpack it slowly.

Suppose we know that if it’s true for n=kn=k, then it’s true for n=k+1n=k+1, for any kk. Then we know that it’s true for k=1k=1, so we know it must be true for k+1=2k+1=2. But we know it’s true for k=2k=2, so it must be true for k+1=3k+1=3. But it’s true for k=3k=3, so it must be true for k+1=4k+1=4. and so on. This hits every positive integer so it must be true for all positive integers.

Let’s try an example.

questions
  1. Prove that any complete graph where every edge is coloured one of two colours has a colour such that the subgraph with edges in that colour is connected. In other words, if in a town, any two towns are connected via either a road or a rail, then you can either get wherever you want using only road, or using only rail.
  2. A person starts at (0,0)(0,0) and takes nn steps, each moving one unit in one of the cardinal directions (north, east, south or west). How many ways are there for the person to reach (a,b)(a,b) (in terms of nn)?
  3. Prove that for any positive integers m,nm,n, there exists NN such that if you colour all the edges on a complete graph on NN vertices with one of nn colours, there exists a monochromatic subgraph of size mm; that is, there exists a set of mm vertices and a single colour AA such that any two of the mm vertices are connected by the same colour AA.
  4. In Lineland there are n≥1n≥1 towns, arranged along a road running from left to right. Each town has a left bulldozer (put to the left of the town and facing left) and a right bulldozer (put to the right of the town and facing right). The sizes of the 2n2n bulldozers are distinct. Every time when a left and right bulldozer confront each other, the larger bulldozer pushes the smaller one off the road. On the other hand, bulldozers are quite unprotected at their rears; so, if a bulldozer reaches the rear-end of another one, the first one pushes the second one off the road, regardless of their sizes.
    Let AA and BB be two towns, with BB to the right of AA. We say that town AA can sweep town BB away if the right bulldozer of AA can move over to BB pushing off all bulldozers it meets. Similarly town BB can sweep town AA away if the left bulldozer of BB can move over to AA pushing off all bulldozers of all towns on its way.
    Prove that there is exactly one town that cannot be swept away by any other one.
  5. Let X1,X2,…,X100X1,X2,…,X100 be a sequence of mutually distinct nonempty subsets of a set SS. Any two sets XiXi and Xi+1Xi+1 are disjoint and their union is not the whole set SS, that is, Xi∩Xi+1=∅Xi∩Xi+1=∅ and Xi∪Xi+1≠SXi∪Xi+1≠S, for all i∈1,…,99i∈1,…,99. Find the smallest possible number of elements in SS.
  6. Let ←← denote the left arrow key on a standard keyboard. If one opens a text editor and types the keys “ab←← cd ←←←← e ←←←← f”, the result is “faecdb”. We say that a string BB is [i]reachable[/i] from a string AA if it is possible to insert some amount of ←←’s in AA, such that typing the resulting characters produces BB. So, our example shows that “faecdb” is reachable from “abcdef”.
    Prove that for any two strings AA and BB, AA is reachable from BB if and only if BB is reachable from AA.
  7. Let m1,m2,…,mnm1,m2,…,mn be a collection of nn positive integers, not necessarily distinct. For any sequence of integers A=(a1,…,an)A=(a1,…,an) and any permutation w=w1,…,wnw=w1,…,wn of m1,…,mnm1,…,mn, define an [i]AA-inversion[/i] of ww to be a pair of entries wi,wjwi,wj with i<ji<j for which one of the following conditions holds:
    ai≥wi>wjai≥wi>wj,
    wj>ai≥wiwj>ai≥wi, or
    wi>wj>aiwi>wj>ai.
    Show that, for any two sequences of integers A=(a1,…,an)A=(a1,…,an) and B=(b1,…,bn)B=(b1,…,bn), and for any positive integer kk, the number of permutations of m1,…,mnm1,…,mn having exactly kk AA-inversions is equal to the number of permutations of m1,…,mnm1,…,mn having exactly kk BB-inversions.
  8. Let n>1n>1 be a positive integer. Consider a n×n×nn×n×n cube, with each unit cube given a colour. For each n×n×1n×n×1 block, we consider the set of colours present in that block. This gives us 3n3n sets, split into 33 groups by orientation. It happens that for any set in a group, the same set appears in all other groups. Find the maximum number of colours possible, in terms of nn.
  9. In Planetland, given any pair of Towns A and B, you can either get from A to B or (exclusive or, so not and) from B to A, using either Red Coaches or (really, xor, i.e not and) Blue Coaches. Prove that there exists a town X such that for any town Y, there exists a colour K it’s possible to get from X to Y using only K Coaches. note: K is Red xor Blue.
  10. Let a1,a2,…,ana1,a2,…,an be distinct positive integers and let MM be a set of n−1n−1 positive integers not containing s=a1+a2+…+ans=a1+a2+…+an. A grasshopper is to jump along the real axis, starting at the point 00 and making nn jumps to the right with lengths a1,a2,…,ana1,a2,…,an in some order. Prove that the order can be chosen in such a way that the grasshopper never lands on any point in MM.
