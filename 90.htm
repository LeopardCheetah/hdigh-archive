infinite integer sequences

An integer sequence is a sequence of integers.

Important things to note about integer sequences: they are discrete; in other words, if they are decreasing after a certain point, you have to hit 0 (and go negative) eventually. Another thing is if the sequence is bounded (after a point), then there exist arbitrarily long sequences which occur infinitely often. In other words, if |ai|<M|ai|<M eventually, then for any nn, there exists a sequence b1,b2,b3,…,bkb1,b2,b3,…,bk such that there are infinitely many nn such that an+i=bian+i=bi for the appropriate ii. This is e.g useful for proving that a recurrence [note: i’ll use this in the sense of “a term is a function of the previous NN terms where NN is fixed”] repeats: if a term is dependent only on the previous three terms, then if you can prove that the entire sequence is bounded, then it has to repeat [is periodic].

Let’s see an example.

Suppose an=|an−1−an−2|an=|an−1−an−2| is an integer sequence. Prove that it repeats.

Note that the sequence is strictly positive from a3a3 onwards, and that by definition, ai≤max(ai−1,ai−2)ai≤max(ai−1,ai−2). Thus aiai is bounded, and since each aiai is a function of the previous two aiai, it also repeats.

Remember with sequences, induction is always a possibility. It’s actually a very powerful technique for sequences in general.

Note that if you consider a recurrence mod mm, it has to repeat.

Prove that, for any k∈Nk∈N, there exists an nn such that k∣Fnk∣Fn, the nnth Fibonacci number.

Note that if we “extend” the sequence backwards, we get F0=0F0=0 (rigorise this! Hint: construct a new sequence F′F′ and show F′=FF′=F for all positive integer points). Then consider the sequence modkmodk; it has to repeat, and therefore we’re done.

One final word of advice: if you get a question and it defines a sequence, write down the first few terms! You might be surprised at how easy the pattern is.
